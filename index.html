Google Sheets Bağlantısı

Veriler Google Sheets'te saklanır. Script URL'nizi buraya girin.

Apps Script Kodu

Aşağıdaki kodu Google Apps Script'e yapıştırın, deploy edin ve URL'yi yukarıya girin: 

const SHEET_NAME = 'data';

function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const raw = sheet.getRange('A1').getValue();
  const data = raw ? JSON.parse(raw) : { alimlar: [], odemeler: [] };
  return ContentService.createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  sheet.getRange('A1').setValue(e.postData.contents);
  return ContentService.createTextOutput('OK')
    .setMimeType(ContentService.MimeType.TEXT);
}
